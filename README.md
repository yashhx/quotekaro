# QuoteKaro — prototype

React + Vite app. Installable as a PWA (Add to Home Screen).
Each user's data is stored privately on their own device (localStorage).

## Run locally
```
npm install
npm run dev
```

## Deploy (Vercel or Netlify)
- Framework preset: **Vite**
- Build command: **npm run build**
- Output directory: **dist**

That's it — connect this folder/repo and deploy.
